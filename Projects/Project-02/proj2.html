Originality

As a designer, originality is very important, because the value of a designer is largely reflected in originality. If we don't go to create, then our work is actually meaningless in a large sense.

As a designer, we may all start with imitation in our work at the beginning. However, the more we imitate, the more our uniqueness will disappear simultaneously.

If we imitate too much, such an imitation will become plagiarism, and this plagiarism will slowly affect our creativity.

Gradually, we will become machines that only copy and can no longer create valuable works.
